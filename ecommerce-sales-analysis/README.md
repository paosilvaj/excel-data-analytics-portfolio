# Análisis de Ventas E-commerce (Excel)

## Descripción
Análisis de datos de ventas de e-commerce que incluye limpieza de datos, cálculo de KPIs, uso de tablas dinámicas y creación de un dashboard.

## Tareas principales
- Limpieza y estandarización de datos
- Análisis de ventas utilizando fórmulas de Excel (SUMAR.SI, PROMEDIO.SI, CONTAR.SI)
- Creación de tabla dinámica y dashboard
- Generación de insights y recomendaciones de negocio

## Herramientas
- Microsoft Excel

## Insights
- La categoría Electrónica generó los mayores ingresos
- El canal Web superó al canal App en compras de mayor valor

# E-commerce Sales Analysis (Excel)

## Description
Analysis of e-commerce sales data including data cleaning, KPI calculation, pivot tables, and dashboard creation.

## Key Tasks
- Data cleaning and standardization
- Sales analysis using Excel formulas (SUMIF, AVERAGEIF, COUNTIF)
- Pivot table and dashboard creation
- Business insights and recommendations

## Tools
- Microsoft Excel

## Insights
- Electronics category generated the highest revenue
- Web channel outperformed App in high-value purchases
